`app` - главная директория проекта. Точка входа в проект
`features` - директория хранящие бизнес логику страниц

`packages` - папка с общим кодом (web + mobile)
`packages/design-tokens` - набор дизайн системы (на будущее)
`packages/entities` - общие сущности
`packages/features` - общая логика
`packages/functions` - вспомогательные функции
`packages/shared` - общие компоненты (в будущем измениться) (не трогать)
`packages/shared-components` - общие большие UI компоненты (шапка сайта, карточки и т.д.)
`packages/shared-ui` - общие UI компоненты (поле ввода, кнопка, модалка и т.д.)
`packages/widgets` - компоненты части страниц, которые содержат сложную логику
