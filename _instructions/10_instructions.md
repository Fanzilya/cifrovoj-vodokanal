# Правила

Переходы по страницам происходят по url
url - это текст в панельке поисковика (http://hydrig.gsurso.ru)

--- 

Когда пользователь переходит на страницу с меню, то url меняется на (http://hydrig.gsurso.ru/menu-moduls)
Чтобы реализовать в мобилке переход на страницу меню, нужно:
1) создать папку с именем url (menu-moduls)
2) создать в папке файл index.tsx 

---



# Вёрстка страницы
1) В промпт закидываешь код web файла страницы
2) Пишешь требования:
    - перепиши код под React Native Expo
    - для стилей используй nativewind
3) В коде web файла есть import. Проверь правильно ли подключены
4) если подчёркнуто красным, значит подключены не правильно. Тогда нужно:
    1) удалить строчку import
    2) найти где использовался импортированный элемент
    3) удалить последную букву и вернуть её, тогда появиться подсказка редактора
    4) выбрать нужный комопнент
5) после правильного подключения нужно зайти в компонент, который подключили через import и отправить нейронке на проверку совместимости с React Native
6) если нужны изменения, то вносишь изменения от нейронки и проверяешь работает ли
