# Правила

Переходы по страницам происходят по url
url - это текст в панельке поисковика (http://hydrig.gsurso.ru)

--- 

Когда пользователь переходит на страницу с меню, то url меняется на (http://hydrig.gsurso.ru/menu-moduls)
Чтобы реализовать в мобилке переход на страницу меню, нужно:
1) создать папку с именем url (menu-moduls)
2) создать в папке файл index.tsx
