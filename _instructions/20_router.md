`+not-found` — 404, перехватывает все несуществующие маршруты
`+html` — кастомизация HTML-шаблона (только для web)
`+native-intent` — обработчик глубоких ссылок (диплинков)
`+middleware` — промежуточный слой (аутентификация, редиректы)




# Основные компоненты навигации:
`Stack` — для вывода дочерних элементов. Используется в _layout.tsx (аналог createStackNavigator) 
`Tabs` — переключение панелей без перезагрузки
`Drawer` — для бокового меню
`<Link href="/path">` - компонент ссылка


# Навигационные хуки
`useRouter()` — для программной навигации (push, back, replace)
`usePathname()` — получить текущий путь
`useLocalSearchParams()` — получить параметры URL (получаем данные из [name] в директории)
